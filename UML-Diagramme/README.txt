UML Diagramme erstellen
=======================
- Download UML/Interlis Editor von http://www.interlis.ch/interlis2/download23_d.php
- Auspacken des jar-Files umleditor.jar
- programm starten mit java -jar <pfad_nach>/umleditor.jar
- File modelle-umleditor.uml öffnen (Datei -> Öffnen)
- Die Modelle werden im Navigationsbaum links geladen
- Vom gewünschten Modell zum Eintrag "Klassen" navigieren und Doppelklicken -> Klassendiagramm wird geöffnet

- Neue Modelle importieren mit Werkzeuge -> Interlis -> Importieren
